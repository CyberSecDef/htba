# Glossary

## Terms

**ACK** - Acknowledgment packet in TCP handshake

**ACL** - Access Control List

**CIFS** - Common Internet File System, a dialect of SMB

**IDS** - Intrusion Detection System - monitors traffic passively

**IPS** - Intrusion Prevention System - blocks attacks actively

**NSE** - Nmap Scripting Engine

**RTT** - Round-Trip Time

**SMB** - Server Message Block protocol

**SYN** - Synchronize packet in TCP handshake

**TCP** - Transmission Control Protocol

**UDP** - User Datagram Protocol
